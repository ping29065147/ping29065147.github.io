# 第三章 Chapter 3 Start and End

指令說明 instruction explanation

  1. (start)string1 = string2 、 (end)string1 = string2 若 string1 在字串的開頭、結尾，將 string1 替換成 string2
    if string1 is at the start / end of string, replace string1 with string2
  2. string1 = (start)string2 、 string1 = (end)string2 找到最左方的 string1 ，將之刪除，並在字串的開頭、結尾加入 string2
    find the left most string1 , delete it, and add string2 at the start / end of string
